# StudyHub - LEARNLINK

## Project Description
LEARNLINK is a virtual classroom platform that combines course management with live video sessions.

## Features Planned
- User registration (Students & Teachers)
- Course enrollment system
- Virtual classrooms with video integration
- Attendance tracking
- Assignment submission
