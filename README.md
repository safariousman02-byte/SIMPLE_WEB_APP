LearnLink is a full-stack web application for online education. Students can browse courses, register accounts, login, and track their learning progress.

- **Frontend**: React.js, HTML5, CSS3, JavaScript
- **Backend**: Node.js, Express.js
- **Database**: MySQL
- **Testing**: Jest, Supertest
- **Containerization**: Docker, Docker Compose
- **Version Control**: Git, GitHub

## Installation & Setup

### Prerequisites
- Node.js (v18 or higher)
- MySQL (v8.0 or higher)
- Git
